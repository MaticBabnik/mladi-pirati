# Mladi Pirati

Uradna spletna stran Mladih Piratov — podmladka Piratske stranke Slovenije.

## Stack

- [Astro](https://astro.build) — static site generator
- JetBrains Mono — monospace typography
- Variable typographic ASCII — canvas-rendered background
- Vanilla JS — no frameworks

## Development

```sh
npm install
npm run dev
```

Dev server runs at `localhost:4321`.

## Build

```sh
npm run build
npm run preview
```
